## 🚀 Where to Start?

=== "I'm completely new to AI agents"
    Start with **Lab 001** → [What are AI Agents?](labs/lab-001-what-are-ai-agents.md)  
    Then follow the conceptual labs (L50) before picking a path.

=== "I know AI basics, want hands-on labs"
    Head to **[Lab 013 — GitHub Models](labs/lab-013-github-models.md)** or  
    **[Lab 014 — SK Hello Agent](labs/lab-014-sk-hello-agent.md)**.  
    Both use free GitHub Models — no credit card needed.

=== "I have an Azure subscription"
    Jump to **[Lab 030 — Foundry Agent Service + MCP](labs/lab-030-foundry-agent-mcp.md)**  
    for a full end-to-end cloud agent experience.

=== "I want to connect tools to agents (MCP)"
    Start with **[Lab 012 — What is MCP?](labs/lab-012-what-is-mcp.md)**  
    then build your own server in **[Lab 020 (Python)](labs/lab-020-mcp-server-python.md)** or **[Lab 021 (C#)](labs/lab-021-mcp-server-csharp.md)**.
