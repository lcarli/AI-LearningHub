## 🚀 Par où commencer ?

=== "Je suis totalement nouveau dans les agents IA"
    Commencez par le **Lab 001** → [Que sont les agents IA ?](labs/lab-001-what-are-ai-agents.md)  
    Puis suivez les labs conceptuels (L50) avant de choisir un parcours.

=== "Je connais les bases de l'IA, je veux des labs pratiques"
    Rendez-vous au **[Lab 013 — GitHub Models](labs/lab-013-github-models.md)** ou  
    **[Lab 014 — SK Hello Agent](labs/lab-014-sk-hello-agent.md)**.  
    Les deux utilisent GitHub Models gratuitement — aucune carte bancaire nécessaire.

=== "J'ai un abonnement Azure"
    Passez directement au **[Lab 030 — Foundry Agent Service + MCP](labs/lab-030-foundry-agent-mcp.md)**  
    pour une expérience complète d'agent cloud de bout en bout.

=== "Je veux connecter des outils aux agents (MCP)"
    Commencez par le **[Lab 012 — Qu'est-ce que MCP ?](labs/lab-012-what-is-mcp.md)**  
    puis construisez votre propre serveur dans le **[Lab 020 (Python)](labs/lab-020-mcp-server-python.md)** ou le **[Lab 021 (C#)](labs/lab-021-mcp-server-csharp.md)**.
