- ✅ Multi-agent systems with **AutoGen** (orchestrator + specialist agents)
- ✅ Production patterns: retry, circuit breaker, cost control
- ✅ Custom **LangChain** chains with tool calling
- ✅ End-to-end evaluation pipeline
