# ⚙️ Trilha Agentes Pro Code

<span class="level-badge level-200">L200</span> <span class="level-badge level-300">L300</span> <span class="level-badge level-400">L400</span>

Para desenvolvedores que desejam controle total sobre a arquitetura de seus agentes. Vá além dos serviços gerenciados e construa agentes prontos para produção com frameworks open-source e o Azure AI SDK.

---

## O que Você Vai Construir

- ✅ Sistemas multi-agente com **AutoGen** (orquestrador + agentes especialistas)
- ✅ Padrões de produção: retry, circuit breaker, controle de custos
- ✅ Cadeias **LangChain** personalizadas com chamada de ferramentas
- ✅ Pipeline de avaliação de ponta a ponta

## Frameworks Abordados

| Framework | Linguagem | Melhor para |
|-----------|-----------|-------------|
| **AutoGen** | Python | Orquestração de conversas multi-agente |
| **Semantic Kernel** | Python / C# | Agentes baseados em plugins, stack Microsoft |
| **LangChain** | Python / JS | Cadeias de agentes de propósito geral |
| **Azure AI SDK** | Python / C# | Desenvolvimento de agentes nativo do Azure |
