# 👥 Agent Builder — Trilha Microsoft Teams

<span class="level-badge level-100">L100</span> <span class="level-badge level-200">L200</span> <span class="level-badge level-300">L300</span>

Construa agentes de IA que vivem dentro do **Microsoft Teams** — o hub diário para milhões de trabalhadores. De agentes Copilot Studio sem código a bots pro-code com a Teams AI Library.

---

## O que Você Vai Construir

- ✅ Um agente de perguntas e respostas com **Copilot Studio** (sem código)
- ✅ Um bot conversacional com a **Teams AI Library** (pro-code, Node.js ou C#)
- ✅ Um **agente com motor personalizado** que se conecta aos seus próprios dados

## Qual ferramenta usar?

| Cenário | Ferramenta |
|---------|------------|
| Sem código / low-code | Copilot Studio |
| Lógica personalizada, controle de código | Teams AI Library |
| Estender um M365 Copilot existente | Copilot Studio + conector personalizado |
| Raciocínio complexo em múltiplas etapas | Teams AI Library + Semantic Kernel |

---

## Pré-requisitos

- Tenant M365 Developer gratuito → [developer.microsoft.com/microsoft-365/dev-program](https://developer.microsoft.com/microsoft-365/dev-program)
- Para a Teams AI Library: Node.js 18+ ou .NET 8
