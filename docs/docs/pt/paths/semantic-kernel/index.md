# 🧠 Trilha Semantic Kernel

<span class="level-badge level-100">L100</span> <span class="level-badge level-200">L200</span> <span class="level-badge level-300">L300</span> <span class="level-badge level-400">L400</span>

Semantic Kernel (SK) é o SDK open-source da Microsoft para construir agentes e aplicações de IA em **Python**, **C#** e **Java**. Oferece uma rica camada de abstração sobre LLMs com suporte para plugins, memória, armazenamento vetorial, chamada automática de funções e orquestração multi-agente.

---

## O que Você Vai Construir

- ✅ Seu primeiro agente SK com **GitHub Models (gratuito)**
- ✅ **Plugins** (funções nativas + conectores OpenAPI)
- ✅ **Memória vetorial** para agentes com reconhecimento de contexto
- ✅ **Chamada automática de funções** com planejadores
- ✅ **Orquestração multi-agente** com padrão orquestrador + trabalhador
