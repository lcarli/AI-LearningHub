## 🚀 Por Onde Começar?

=== "Sou completamente novo em agentes de IA"
    Comece com o **Lab 001** → [O que são Agentes de IA?](labs/lab-001-what-are-ai-agents.md)  
    Depois siga os laboratórios conceituais (L50) antes de escolher um caminho.

=== "Conheço o básico de IA, quero laboratórios práticos"
    Vá para o **[Lab 013 — GitHub Models](labs/lab-013-github-models.md)** ou  
    **[Lab 014 — SK Hello Agent](labs/lab-014-sk-hello-agent.md)**.  
    Ambos usam GitHub Models gratuito — sem necessidade de cartão de crédito.

=== "Tenho uma assinatura do Azure"
    Pule para o **[Lab 030 — Foundry Agent Service + MCP](labs/lab-030-foundry-agent-mcp.md)**  
    para uma experiência completa de agente na nuvem de ponta a ponta.

=== "Quero conectar ferramentas a agentes (MCP)"
    Comece com o **[Lab 012 — O que é MCP?](labs/lab-012-what-is-mcp.md)**  
    depois construa seu próprio servidor no **[Lab 020 (Python)](labs/lab-020-mcp-server-python.md)** ou **[Lab 021 (C#)](labs/lab-021-mcp-server-csharp.md)**.
